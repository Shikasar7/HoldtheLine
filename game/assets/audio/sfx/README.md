## Delivery processing

The Kenney-derived clips are stereo OGG/Vorbis at 44,100 Hz. During integration each decoded sample peak was
measured with FFmpeg `volumedetect`; gain was applied and the encoded result re-measured so its peak
does not exceed the `-6 dBFS` delivery ceiling (`-ar 44100 -ac 2 -c:a libvorbis -q:a 5`).

The original synthesized cues are mono PCM WAV at 44,100 Hz and are intentionally kept uncompressed.
`SfxBank` checks `<key>.ogg` and then `<key>.wav`. If a file is
missing or Godot cannot import it, it retains the former synthesized clip for that key, so incomplete exports degrade safely rather
than becoming silent.
