## v0.7.4 · 2026-07-21 · 教团法术改版与秘密体系（平衡补丁 #4）

> 本次是**教团阵营的大改版 + 一批全新机制**（规则版本 0.8.1 → **0.9.0**），联机需服务端与客户端**同步更新到本版本**后才能对局；单机 / 人机不受影响。完整数值见 `balance/BALANCE-CHANGELOG.md` 补丁 #4。此前待发布的「UI 全面美化改版」也随本版一并上线（见文末）。

### 教团法术改版（核心）
- **直伤要选位了**：教团的点伤不再"隔空全场点名"。战吼类改为**锚·N**（以随从落地格为中心，曼哈顿 N 格内选目标），指令类改为**引导·N**（施放前先选一个友方随从当"引导者"，目标须在引导者 N 格内）。
- **引导者差异化**：由谁引导会改变效果——**焰术学徒**引导 → 该薪炎指令伤害 +1；**熔岩巨灵**引导 → +2；**晚祷领唱**引导 → 该薪炎指令费用 -1（下限 1 费）。这三张随从的旧战吼/触发已换成上述"引导加成"。
- **薪炎伤害 + 加深 / 蓄能**：教团的伤害与灼蚀统一为**薪炎**类型。**焰跃术士**战吼改为**蓄能 2**（你的下一张薪炎伤害指令 +2）。
- **归魂**：**灼誓狂徒**新增——你的回合内每有友方随从死亡，获得 1 点本回合辉尘（每回合最多 2 次）。
- **自体成长设上限**：灰烬侍徒 / 烬眼先知 / 烬火唱徒的"每施放指令自身成长"每回合最多叠 **2 次**；黄昏圣咏·奥菲兰不受此限。
- **卡牌逐张**：星火→引导·2、焰鞭→引导·2 双模式（打敌 3 薪炎 / 消灭友方并把其攻血转移给另一友方）、献祭 2→**1 费**、灼痕烙印→引导·2、晚祷烈焰 / 焰幕 / 烬蚀之列→引导·3、燔火→**引导：5 发 1 点薪炎随机分配**（每点加深/蓄能 +1 发）、烬爆蛾→1 费 0/2 亡语 3 薪炎、焰击术士→锚·2；**焚烬之列重做为「燎原」**（7 费，对所有敌方随从 3 点薪炎灼蚀）；**门德改名「薪火回响」**（你每回合第一张薪炎伤害指令以相同目标额外结算一次）；焚世巨灵→你每施放 4 费以上指令后对所有敌方随从造成 1 薪炎；**烬火雏凤/灰烬凤凰免疫薪炎**，雏凤 4 个己方回合后成长为凤凰（受薪炎伤害会加速成长）。

### 新机制 · 秘密与场地
- **烟幕区**：目标格及相邻十字共 5 格，区内随从无法攻击且不反击，走出即解除。
- **烬火陷阱（暗牌）**：在无人格子埋设隐藏陷阱，仅你可见；随从进入即受 3 薪炎灼蚀，现形后火焰持续 2 回合。
- **秘密（暗牌）**：秘密牌打出后进入你的秘密区，对手只看到"存在 N 张暗牌"，看不到内容——陷阱位置与秘密内容由服务端权威保管，不下发给对手。
- **焰誓反制（暗牌）**：下次敌方指令牌选中你的随从时，该指令无效，改为对施放者一方随机随从造成 3 薪炎。
- **潜行**（匿踪）：不能被敌方指令 / 战吼选中，攻击后现形。**法术护体**：抵挡下一次敌方指令 / 战吼效果。

### 新卡（7 张）
- **教团**：熔剑祭士（4 费 2/4，战吼可献祭手牌 2 张指令 → 装备熔岩巨剑 +3 攻 / 射程 2 / 贯穿）、烟幕弹、灰缚（定身）、焰誓反制、烬火陷阱。
- **中立**：匿踪（潜行）、法术护体。

### 中立调整
- **齐射指令**：改为只能打**你半场**的随从。
- **火把杂役**：战吼 → **亡语：对相邻敌方随从造成 1 点伤害**。
- **爆破工兵**：战吼改为**锚·2** 内一个随从 2 伤。

### 规则 / 协议随动（规则版本 0.9.0）
- 引擎新增伤害类型、锚 / 引导定位、加深 / 蓄能、格子状态（烟幕 / 陷阱）、秘密区、成长 / 转化等机制；指令新增引导者 / 二段目标 / 献祭弃牌字段（**加性**，旧对局日志逐字节回放不变）。
- **对手视图剥离隐藏陷阱坐标与秘密内容**（服务端权威，杜绝改客户端窥探）。
- 卡池 163 → **170**；**DataHash 变化**，服务端与客户端必须同版本、同数据。

## （随 v0.7.4 一并上线）UI 全面美化改版与卡面精修

> 纯视觉与体验更新，规则、卡池、数值均无改动。

### 界面焕新
- **全新界面视觉（战地手账风）**：主菜单、对战界面、卡组编辑、结算界面全面换装——钢板浮雕按钮、羊皮纸窗口面板、思源宋体标题大字，以及一套统一的功能图标（手牌 / 牌库 / 辉尘 / 各对战入口 / 换牌 / 设置等），整体质感向"桌面战棋"靠拢。
- **主操作一眼可辨**：每屏唯一的主按钮（开战 / 排位匹配 / 结束回合）改为醒目的金铜色板，和其它钢板按钮一眼分出主次。
- **棋盘部署格**：空格子改为印刻在战桌上的刻痕框，立牌落位更贴合桌面，不再是漂浮的半透明方块。
- **场景切换过渡**：进出对局、返回菜单等切换加入黑场淡入淡出，不再生硬跳切；连点「开战」也不会误触发两次进场。
- **手牌悬停反馈**：鼠标悬停手牌时卡牌轻微上浮放大，更容易看清当前指向的是哪张牌。
- **窗口化二级界面**：人机对战、大厅、登录 / 注册、账号、天梯等页面重做为居中的羊皮纸窗口，排版更聚拢整洁。

### 卡面美术精修
- **卡面构图逐张精修**：此前所有卡面按统一比例硬裁，经常把插画主体的头或脚切掉。现在改为保留完整插画、逐卡手工取景（缩放 + 平移），让每张卡的主体都完整、居中地落在卡框窗口内；对战手牌、检视大图与卡组预览均采用同一套精修取景。

## v0.7.2 · 2026-07-20 · 「守护 / 嘲讽」拆分与防御机制补丁 #2

> 本次含关键词与卡牌调整（规则版本 0.7.0 → 0.8.0），联机需服务端同步更新后才能与旧版本对局；单机/人机不受影响。完整数值见 `balance/BALANCE-CHANGELOG.md` 补丁 #2。

### 关键词调整
- **「守护」正名为「嘲讽」**：原来的「守护」其实是"相邻敌方必须优先攻击它"的嘲讽机制，现更名为**嘲讽**，机制不变。原有守护单位的关键词与文本同步改为嘲讽。
- **新增「福泽」**：与其相邻的友方随从受到的伤害 **-1**（不含自身，可与坚守叠加）。
- **新增「守护」（全新机制）**：与其相邻的友方随从将要受到的伤害，**转移到守护者身上承受**，并享受守护者自己的减伤（坚守/福泽/持盾）。战斗中会以飘字标示：被守护的单位显示「守护-0」，替它挡伤的守护者显示「守护-实际伤害」。

### 卡牌调整
- **壁垒射手**：2/3 → **3/2**。
- **越线者死**：4 费 3 伤 → **3 费 4 伤**。
- **磐石巨像**：嘲讽 → **守护**（保留坚守）。
- **磐石哨卫**：新增**守护**（保留坚守）。
- **移动要塞**：6/8 嘲讽 → **5/10 守护**（保留坚守，更肉）。
- **铁誓圣壁·薇兰蒂**：删除亡语，改为常驻**福泽**（保留嘲讽、持盾）。
- **尖桩拒马 / 装甲路桩**：移除「架设」，现在可以自由移动补位。

## v0.7.1 · 2026-07-20 · 卡面状态指示器与手感优化

> 本次含一处规则调整（规则版本 0.6.0 → 0.7.0），联机需服务端同步更新后才能与旧版本对局；单机/人机不受影响。

### 新功能
- **卡面状态指示器**：场上随从的卡面新增状态指示器，用来直观展示身上的增益/减益状态。首批接入**持盾**——持盾时卡面顶部显示「持盾」标记；持盾**新获得或被打破时，卡面标记会即时更新**（获得时亮起、被消耗时立刻消失），不再出现「盾已破但卡面还显示持盾」的错位。持盾不可叠加，每个单位最多持有一层。

### 玩法调整
- **手牌溢出改为进墓地（规则版本 0.7.0）**：手牌已满（9 张）时再抽到 / 获得的新牌，改为**进入墓地**，不再直接从游戏中烧掉移除。进墓地的指令牌因此可被「火种循环」等取回效果重新利用。（此前 0.5.0 的「超过上限的牌被烧掉」由此调整。）

### 修复
- **随机对手不再总是撞上同阵营**：人机对战选择「随机对手」时，会主动避开与你当前卡组**相同阵营**的对手，随机结果更有变化（仅当没有其它阵营可选时才会出现镜像对局）。

### 修复
- 修复静默自动登录 / 断线重连后，账号有时被错误显示为「游客」或误报「已登出」的问题。
- 修复压力潮汐致死预警的措辞与触发时机：仅在确实无法避免时才发红色预警，不再虚惊一场。
- 修复账号偏好数据在后台并发写入时可能损坏的问题。
- 修复在登录 / 改名等表单操作过程中返回上一页偶发崩溃的问题。

## v0.5.0 · 2026-07-19 · 卡组体验与更新通道

### 平衡与规则调整（规则版本 0.5.0）
- **军令硬币**使用后从游戏中移除，不再进入墓地——教团的「从墓地取回指令牌」类效果（火种循环）无法再把硬币捞回来反复触发。
- **手牌上限从 10 张调整为 9 张**：后手开局 6 张 + 军令硬币仍有余量，同时让无限囤牌有真实代价。超过上限抽到的牌仍会被烧掉。

### 玩法优化
- **随从战吼判定时机优化**——「先上随从，再判战吼」：需要指定目标的战吼，如果场上根本没有合法目标，随从照常上场、战吼自动落空，不再因为「选不到目标」而整张卡卡住打不出。场上有合法目标时，仍然必须指定。（指令牌与领袖技能不适用此规则：无目标打出仍算废牌，保持严格。）

### 新功能
- **人机对战改版**：单一入口，选「我的卡组 × 难度（简单/普通/困难）× 对手卡组（可随机）」直接开战。
- **人机 AI 优化**：困难档启用**前瞻搜索 AI**——它会预演走完这步、对手回应之后的局面再决策；简单/普通档用更保守的即时启发式，三档手感明显拉开差距。
- **账号系统**：支持用户名 + 密码注册/登录，进度可跨设备找回。
- **卡组口令**：一键复制卡组口令分享给朋友，对方粘贴即可导入整套卡组。
- **自动更新**：有新版本时主菜单出现更新横幅，点击即可下载更新（安装版自动换新并重启，绿色版跳转下载页）。

### 体验与操作优化
- **菜单优化**：主菜单角落显示当前版本号；卡组选择界面把「自己的卡组」排在内置卡组前面，默认选中上一把使用的卡组（没打过则选最近编辑的），鼠标悬停卡组即可预览领袖与完整构成。
- **卡组命名**：新建卡组自动编号（我的卡组1、我的卡组2……），改名/复制/导入也不会再出现重名。
- **登录/连接流程优化**：连接服务器被拒时，会明确告知原因（客户端版本过旧、本机卡表与服务器不一致等）并直接引导更新，不再只给一句笼统的「连接失败」；顺带修好了握手偶发卡住超时的老问题，连接更干脆。
- **对局中返回优化**：联机对局点「返回菜单」不再让你直接消失掉线，而是**确认后视为投降并留在原地观看结算**；断线彻底失败等特殊情况才直接退出，服务端也有兜底。
- **卡牌操作优化**：新增「取消」按钮（也可按 Esc）随时退出选择；正在指定目标的手牌会**抬起放大高亮**，让你始终看清「现在在结算哪张牌」。
- **音效升级**：13 条正式音效素材上线，替换原先的程序合成音。
- **架设强化表现**：「架设 +1」类效果命中时有浮字提示和关键词说明，伤害来源一目了然。

### 修复
- 修复选择卡组时可能出现的重名与默认值缺失问题（详见上方卡组命名/菜单优化）。
