# 守线 · Hold the Line

> 一款 1v1 联机卡牌对战游戏：经典的费用与回合节奏，叠上一块 5×4 的共享网格战场。随从不再是"摆在桌上的数值"，而是需要**部署、推进、卡位、包抄**的棋子——把兵推过对方底线，才打得到对方本体。

---

## 这是个什么游戏？

大多数卡牌游戏里，"站位"要么不存在，要么只有一维。《守线》把站位做成了**二维网格**，让"空间"成为和"费用"、"卡差"并列的第三种资源：

```
        敌方本体 (25 HP)
  ┌────┬────┬────┬────┬────┐
  │    │    │    │    │    │  ← 敌方底线行（推到这里才能打脸）
  ├────┼────┼────┼────┼────┤
  │    │    │    │    │    │  ← 敌方前线行
  ├────┼────┼────┼────┼────┤
  │    │    │    │    │    │  ← 我方前线行
  ├────┼────┼────┼────┼────┤
  │    │    │    │    │    │  ← 我方底线行（随从只能部署在这里）
  └────┴────┴────┴────┴────┘
        我方本体 (25 HP)
```

- **回合节奏是熟悉的**：费用水晶每回合 +1（上限 10），抽一张牌，打牌、行动、交回合。后手玩家多一张起手牌和一枚"军令"硬币——硬币可以换 1 费，也可以换 1 格额外移动。用空间补偿代替纯费用补偿，这是本作的味道。
- **随从要自己走过去**：每个随从每回合 1 移动 + 1 攻击，上下左右不可斜走，**任何随从都不可穿越**（友方也不行）。一个 1/5 的白板在这里是货真价实的路障。
- **伤脸必须靠推线**：只有站上敌方底线行的随从才能攻击对方本体，直伤法术刻意做得又少又贵。胜利不是抽出来的，是一格一格推出来的。
- **龟缩合法，但不免费**：第 8 回合起"压力潮汐"启动——敌方半场没有你随从的一方，领袖每回合流血，且伤害逐轮递增。守是战术，缩是自杀。
- **一局 8~12 分钟**：决策密度高、操作量低，打完一局正好够泡面变凉。

## 玩法特色

- **以空间为目标的法术**："选择一列，该列所有敌方随从受到 2 点伤害"——行、列、格子、半场都能当法术目标，清场先看阵型。
- **和棋盘对话的关键词**：嘲讽逼你贴脸、守护替相邻队友挡伤、射程隔墙点名、冲锋疾行调速……每个关键词都在改变某块格子的价值。
- **战术建筑**：路障、哨塔、陷阱直接摆在格子上，把"这块地是我的"写成规则。
- **没有硬性克制表**：四个阵营互有胜负手，也各有反制的招。速攻还是坚守，堡垒还是法术，任由你选择。

## 四大阵营

| 阵营 | 风格 | 怎么玩 |
|------|------|--------|
| **铁誓军团** (Iron Vow) | 墙 | 嘲讽、守护、战术建筑管够，先把你耗到没脾气，再一波推线 |
| **荒野游群** (The Wildpack) | 风 | 冲锋、疾行、换位跳跃，多线开花，让防线顾此失彼 |
| **黄昏教团** (The Duskweavers) | 焰 | 以格子和行列为目标的范围法术，场面可以输，一套连锁清一列 |
| **掘世匠会** (The Undervault) | 械 | 射程与炮台，前排肉盾护住后排炮线，阵型摆对了仗就赢了一半 |

外加中立卡池，当前卡池 160+ 张，30 张构筑（1 阵营 + 中立），持续平衡更新中（平衡日志见 [balance/BALANCE-CHANGELOG.md](balance/BALANCE-CHANGELOG.md)）。

## 现在能玩到什么

- **联机对战**：在线匹配（WebSocket + 服务端权威规则演算，没有人能改内存作弊）
- **人机对战**：内置多套 AI 卡组，随机对手会主动避开镜像对局
- **卡组构筑**：完整的构筑器，支持排序、命名、多套卡组管理
- **自动更新**：客户端启动自动检查新版本，Windows 安装包一键升级

前往 [Releases](https://github.com/Shikasar7/HoldtheLine/releases) 下载最新 Windows 客户端。

## 技术与仓库结构

客户端 **Godot 4 (C#)**，服务端 **.NET / C#**，规则引擎为客户端与服务端共享——同一份 `HoldTheLine.Rules` 同时驱动单机演算和服务端裁决。

```
game/       Godot 客户端（场景、脚本、卡牌数据）
src/        .NET 解决方案
  ├─ HoldTheLine.Rules        规则引擎（含单元测试）
  ├─ HoldTheLine.Server       联机服务端（含单元测试）
  ├─ HoldTheLine.Net          协议层
  ├─ HoldTheLine.BotClient    机器人客户端
  └─ HoldTheLine.Sim          平衡模拟器
balance/    平衡补丁日志
scripts/    构建与发布脚本
```

玩家向更新记录见 [CHANGELOG.md](CHANGELOG.md)。
