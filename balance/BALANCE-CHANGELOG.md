## 2026-07-20 · 平衡补丁 #2 (Rules 0.8.0)

### 关键词体系 — 拆分「守护」,新增两个防御机制

| 关键词 | 改动 |
| --- | --- |
| 守护(旧) | 更名为 **嘲讽**(机制不变:相邻敌方随从必须优先攻击它)。「守护」这一名字被释放出来给下面的新机制。 |
| 福泽(新) | **与其相邻的友方随从受到的伤害 -1**(不含自身,可与坚守叠加)。 |
| 守护(新) | **与其相邻的友方随从将要受到的伤害,转移到守护者身上承受**,并按守护者自身的减伤(坚守/福泽/持盾)结算;守护者被转移来的伤害不再二次转移(无循环)。 |

- 旧「守护」实为炉石式嘲讽,正名后语义更清晰;新「守护」是真正的"替队友挡伤"护卫机制,和铁誓/匠会的防御主题契合。
- 结算顺序:守护转移 → 坚守 → 福泽 → 持盾 → 扣血。灼蚀(sear)仍只无视坚守,福泽照常生效。

### 铁誓 (Iron Vow)

| --- | --- |
| 壁垒射手 | 2/3 → **3/2**(射程 2 不变) |

| 磐石哨卫 | 坚守 → **守护,坚守**(新增守护) |
| 铁誓圣壁·薇兰蒂 | 删除亡语(相邻友方获得嘲讽)→ **新增福泽**(嘲讽,持盾,福泽) |
