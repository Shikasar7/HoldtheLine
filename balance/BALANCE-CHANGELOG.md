# Hold the Line — 平衡补丁日志

> 本文档按时间线记录每一次数值/机制平衡调整。每个补丁一节，标题为 `日期 · 补丁编号 (RulesVersion)`。
> 新补丁追加在最上方。规则引擎语义变化会同步提升 `RulesInfo.Version`,需要重新部署服务端。

## 2026-07-20 · 平衡补丁 #3 (Rules 0.8.1)

### 先后手平衡 — 削弱后手开局抽牌

| 对象 | 改动 |
| --- | --- |
| 后手开局手牌 | 先手 4 张 + 后手 **6 → 5** 张（额外抽牌 +2 → **+1**）；军令硬币保留不变 |

- 后手同时"多抽 2 张 + 军令硬币"后，后手胜率偏高。此前把额外抽牌从 +1 提到 +2 是为补偿先手优势，但叠加硬币后补偿过头。
- 本补丁把额外抽牌回退到 +1（后手开局 5 张 + 硬币，共 6 张手牌），先手 4 张 + 回合首抽 = 5 张。硬币不动，后手仍保留节奏补偿，只是牌差收窄。

### 引擎/协议随动 (Rules 0.8.1)

- 仅改 `MatchConfig.OpeningHandSecond` 默认值 6 → 5，无新字段、无序列化结构变化。
- 旧指令日志里已序列化的 `OpeningHandSecond` 保持原值，历史对局回放逐字节不变。
- **RulesVersion 0.8.0 → 0.8.1**，需服务端+客户端同版本重新部署（与仍待部署的平衡补丁 #1、#2 一并上线）。

## 2026-07-20 · 平衡补丁 #2 (Rules 0.8.0)

### 关键词体系 — 拆分「守护」,新增两个防御机制

| 关键词 | 改动 |
| --- | --- |
| 守护(旧) | 更名为 **嘲讽**(机制不变:相邻敌方随从必须优先攻击它)。「守护」这一名字被释放出来给下面的新机制。 |
| 福泽(新) | **与其相邻的友方随从受到的伤害 -1**(不含自身,可与坚守叠加)。 |
| 守护(新) | **与其相邻的友方随从将要受到的伤害,转移到守护者身上承受**,并按守护者自身的减伤(坚守/福泽/持盾)结算;守护者被转移来的伤害不再二次转移(无循环)。 |

- 旧「守护」实为炉石式嘲讽,正名后语义更清晰;新「守护」是真正的"替队友挡伤"护卫机制,和铁誓/匠会的防御主题契合。
- 结算顺序:守护转移 → 坚守 → 福泽 → 持盾 → 扣血。灼蚀(sear)仍只无视坚守,福泽照常生效。

### 铁誓 (Iron Vow)

- 磐石巨像/磐石哨卫成为铁誓的核心护卫单位,替相邻脆皮扛伤;薇兰蒂改为常驻减伤光环,活着就能护住整片阵线。

### 引擎/协议随动 (Rules 0.8.0)

- 新增关键词 `blessing`(福泽)、`guardian`(守护);旧 `guard` 关键词统一改名为 `taunt`(嘲讽),全部卡表数据随之更新。
- `UnitDamagedEvent` 新增布尔字段 `GuardRedirect`(加性、向后兼容):守护转移时,被守护单位飘 `守护-0`、守护者飘 `守护-实际伤害`。
- 客户端关键词说明/卡面缩写(嘲/福/护)、卡牌详情面板同步更新。
- **DataHash 与 RulesVersion(0.7.0→0.8.0)均已变化,需服务端+客户端同版本重新部署**(与仍待部署的平衡补丁 #1 一并上线)。

## 2026-07-19 · 平衡补丁 #1 (Rules 0.6.0)

### 教团 (Duskweaver) — 削弱上限,补低费曲线

| 卡牌 | 改动 |
| --- | --- |
| 黄昏圣咏·奥菲兰 | 8 费 6/7 → **9 费 6/6** |
| 烬火唱徒 | 3 费 → **2 费** |
| 灼痕烙印 | 固定 3 点灼蚀 → **随机 2 或 3 点灼蚀**(无视坚守不变) |

- 奥菲兰的指令回收+滚雪球身材是教团后期最强终结点,加费削血推迟其上场节奏并降低站场容错。
- 烬火唱徒降费补偿教团前期曲线,让"指令触发器"更早入场。
- 灼痕烙印引入随机量(引擎新增 `amount_max` 字段,基于对局 RNG,回放确定),点杀 3 血单位不再是确定性交换。

### 铁誓 (Iron Vow) — 增强

| 对象 | 改动 |
| --- | --- |
| 领袖技能 瓦伦丁·铁誓 | 筑垒(2费,授予守护至你的下回合) → **授盾(2费,使一个友方随从获得一次持盾,永久直至被打破)** |

- 持盾直接抵消一次任意伤害,对铁誓的贴脸消耗战比一回合守护更有价值,也和阵营"盾"主题一致。

### 游群 (Wildpack) — 内部平衡:削速攻,强肉搏

| 对象 | 改动 |
| --- | --- |
| 裂谷跃兽 | 疾行 2 → **疾行 1**(跃障保留) |
| 移动后+攻击类触发(循迹幼兽/荒野之魂/月影猎后·祁莲) | **每单位每回合最多触发 2 次**(第 3 步起仍可移动,但不再叠攻;非加攻的移动触发如血迹追猎者的移动扫伤不受限) |
| 围猎 (PackTactics) | 夹击伤害加成 +1 → **+2** |
| 践踏 (Trample) | 重做:近战消灭后可占位 → **近战攻击时,对被攻击单位周围相邻的所有单位(含友方)造成等量(攻击力)伤害,溅射不受反击** |

- 疾行+移动叠攻的速攻链一回合可叠出爆炸攻击力,上限 2 次后速度仍是资源,但不再无限滚。
- 围猎/践踏补偿游群的正面作战能力:围猎奖励包抄站位,践踏使大体型单位成为真正的阵型粉碎机(代价是会误伤自己人,需要走位)。
