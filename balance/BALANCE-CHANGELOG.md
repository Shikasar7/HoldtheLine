# Hold the Line — 平衡补丁日志

> 本文档按时间线记录每一次数值/机制平衡调整。每个补丁一节，标题为 `日期 · 补丁编号 (RulesVersion)`。
> 新补丁追加在最上方。规则引擎语义变化会同步提升 `RulesInfo.Version`,需要重新部署服务端。

## 2026-07-19 · 平衡补丁 #1 (Rules 0.6.0)

### 教团 (Duskweaver) — 削弱上限,补低费曲线

| 卡牌 | 改动 |
| --- | --- |
| 黄昏圣咏·奥菲兰 | 8 费 6/7 → **9 费 6/6** |
| 烬火唱徒 | 3 费 → **2 费** |
| 灼痕烙印 | 固定 3 点灼蚀 → **随机 2 或 3 点灼蚀**(无视坚守不变) |

- 奥菲兰的指令回收+滚雪球身材是教团后期最强终结点,加费削血推迟其上场节奏并降低站场容错。
- 烬火唱徒降费补偿教团前期曲线,让"指令触发器"更早入场。
- 灼痕烙印引入随机量(引擎新增 `amount_max` 字段,基于对局 RNG,回放确定),点杀 3 血单位不再是确定性交换。

### 铁誓 (Iron Vow) — 增强

| 对象 | 改动 |
| --- | --- |
| 领袖技能 瓦伦丁·铁誓 | 筑垒(2费,授予守护至你的下回合) → **授盾(2费,使一个友方随从获得一次持盾,永久直至被打破)** |

- 持盾直接抵消一次任意伤害,对铁誓的贴脸消耗战比一回合守护更有价值,也和阵营"盾"主题一致。

### 游群 (Wildpack) — 内部平衡:削速攻,强肉搏

| 对象 | 改动 |
| --- | --- |
| 裂谷跃兽 | 疾行 2 → **疾行 1**(跃障保留) |
| 移动后+攻击类触发(循迹幼兽/荒野之魂/月影猎后·祁莲) | **每单位每回合最多触发 2 次**(第 3 步起仍可移动,但不再叠攻;非加攻的移动触发如血迹追猎者的移动扫伤不受限) |
| 围猎 (PackTactics) | 夹击伤害加成 +1 → **+2** |
| 践踏 (Trample) | 重做:近战消灭后可占位 → **近战攻击时,对被攻击单位周围相邻的所有单位(含友方)造成等量(攻击力)伤害,溅射不受反击** |

- 疾行+移动叠攻的速攻链一回合可叠出爆炸攻击力,上限 2 次后速度仍是资源,但不再无限滚。
- 围猎/践踏补偿游群的正面作战能力:围猎奖励包抄站位,践踏使大体型单位成为真正的阵型粉碎机(代价是会误伤自己人,需要走位)。

### 引擎/协议随动 (Rules 0.6.0)

- 新增效果字段 `amount_max`:damage/sear 支持 `[amount, amount_max]` 区间随机,每个效果掷一次,走对局确定性 RNG。
- 践踏重做后 `AttackCommand.OccupyCellOnKill` 字段废弃(保留反序列化兼容,解析器忽略)。
- `UnitInstance` 新增 `SelfMovedAtkGainsThisTurn` 计数(回合开始重置),AI 打分同步感知上限。
- 卡表文本、客户端关键词说明(践踏/围猎)同步更新;**DataHash 与 RulesVersion 均已变化,需要服务端+客户端同版本重新部署**。
