## 1. 一句话概念与设计支柱

**一句话概念:**
> 一款 1v1 联机卡牌对战游戏:炉石式的费用与卡牌节奏,叠加一块共享网格战场——随从不再是"摆在桌上的数值",而是需要**部署、推进、卡位、包抄**的棋子。把随从推过对方底线,才能伤害对方本体。

**参照系:** 《炉石传说》(费用曲线、回合节奏、随从交换)+《KARDS》(前线概念、部署与推进)+ 战棋(格子、射程、地形)。

**差异化卖点:** 现有卡牌游戏里"站位"要么不存在(炉石),要么只有一维(KARDS 的前线/后方、昆特的排)。本作把站位升级为**二维网格**,让"空间"成为和"费用"、"卡差"并列的第三种资源。

### 四大设计支柱

每个设计决策都要能回答"它服务于哪根支柱":

| # | 支柱 | 含义 | 反例(要避免的) |
|---|------|------|----------------|
| P1 | **空间即资源** | 一个格子的占领、一条纵列的封锁,应当和一张卡、一点费用一样有价值 | 随从站哪都一样,格子只是装饰 |
| P2 | **推线的张力** | 胜利来自"把兵推过去",玩家每回合都在攻防转换间做取舍 | 双方龟缩互甩法术,地图形同虚设 |
| P3 | **卡牌的爽感不丢** | 保留炉石式"抽到关键牌、打出漂亮 combo"的爽感,战棋深度不以牺牲卡牌爽感为代价 | 变成纯战棋,卡牌沦为兵种购买菜单 |
| P4 | **十分钟一局** | 单局 8~12 分钟,决策密度高但操作量低,适配 PC 与移动端 | 单回合思考 3 分钟、一局半小时 |

## 2. 核心玩法:战场与规则

### 2.1 战场布局(设计基准:5 列 × 4 行)

你最初的提议是 3×5。我做了纸面推演后,**推荐 5 列 × 4 行(共 20 格)作为基准**,理由见下方对比。行的方向是"纵深"(从我方到敌方),列是"横向宽度"。

```
        敌方本体 (25 HP)
  ┌────┬────┬────┬────┬────┐
  │ D1 │ D2 │ D3 │ D4 │ D5 │  ← 敌方底线行(我方随从到达此行可攻击敌方本体)
  ├────┼────┼────┼────┼────┤
  │ C1 │ C2 │ C3 │ C4 │ C5 │  ← 敌方前线行
  ├────┼────┼────┼────┼────┤
  │ B1 │ B2 │ B3 │ B4 │ B5 │  ← 我方前线行
  ├────┼────┼────┼────┼────┤
  │ A1 │ A2 │ A3 │ A4 │ A5 │  ← 我方底线行(我方随从只能部署在此行;敌方随从到达此行可打我)
  └────┴────┴────┴────┴────┘
        我方本体 (25 HP)
```

**尺寸对比分析:**

| 方案 | 优点 | 缺点 | 结论 |
|------|------|------|------|
| 3×5(你的原案) | 极快节奏;部署即接敌 | 纵深只有 1 格中间行,"推进"退化为"抢中路",移动机制没有发挥空间;远程/机动等关键词做不出差异 | 可作为"快速模式"变体保留 |
| **4×5(推荐)** | 每方 2 行纵深:部署行→前线行→敌前线→敌底线,随从需 3 次移动才能触底,攻防节奏有 2~3 个回合的缓冲;5 列宽度足够做侧翼包抄,又不至于铺不满 | 比 3 行稍慢,需要"冲锋/疾行"类关键词调速 | **基准方案** |
| 5×5 / 6×5 | 战棋深度最高 | 前期大量回合浪费在赶路上,违背 P3、P4;移动端屏幕塞不下 | 否决 |

> 4 行还有一个隐性好处:**"我方半场 / 敌方半场"天然二分**,很多关键词和法术可以用"半场"作为作用域(如"我方半场的随从+1生命"),规则表达非常干净。

### 2.2 双方本体(领袖)

- 每位玩家有一个**领袖(Leader)**,25 点生命,位于己方底线之后(不占格子)。
- 领袖自带一个**领袖技能**(2 费,每回合一次,类似炉石英雄技能)与一条**阵营被动**,是阵营身份的载体(见第 5 节)。
- 领袖生命归零 → 对方获胜。无疲劳机制替代方案见 2.7"防拖局设计"。

### 2.3 回合结构

采用炉石式**交替回合制**(而非同时行动),先手后手交替。单回合流程:

1. **回合开始**:费用水晶 +1(上限 10)并全部恢复;抽 1 张牌;触发"回合开始"效果。
2. **主要阶段**(自由顺序,可穿插):
   - **打出卡牌**:随从卡部署到**己方底线行的空格**;法术/装备按卡面目标施放。
   - **随从行动**:每个随从每回合拥有 **1 移动 + 1 攻击**(先后顺序不限,可只用其一)。
   - **领袖技能**:消耗 2 费发动。
3. **回合结束**:触发"回合结束"效果,回合交给对方。

**先手/后手平衡**:后手玩家开局多 1 张手牌,并获得一枚**"军令"硬币**(0 费法术:本回合+1费,*或* 令一个随从本回合可额外移动 1 格——二选一)。第二个选项是本作特色:用**空间补偿**代替纯费用补偿。

### 2.4 移动规则

- 默认移动力 1:每回合可向**上下左右**相邻空格移动 1 格(不可斜向,保持格子价值清晰——P1)。
- **不可穿越**任何随从(友方也不行)——这让"身体卡位"成为核心防守手段(P1、P2)。
- 刚部署的随从有**"集结中"**状态(即召唤失调):当回合不能移动、不能攻击,除非有〔冲锋〕〔快速部署〕等关键词。
- 移动不是强制的;龟缩是合法策略,但见 2.7 防拖局设计。

### 2.5 战斗规则

- **近战(默认)**:可攻击**上下左右相邻**的敌方随从。攻击为炉石式**互殴**:双方同时按攻击力互相扣血。
- **远程〔射程 N〕**:可攻击同一**列**或同一**行**上、直线距离 ≤ N 且**中间无阻挡**的敌人;远程攻击**不吃反击**(近战摸不到它时),但远程随从被近战贴脸攻击时正常互殴。
- **攻击本体**:随从位于**敌方底线行**时,攻击阶段可以选择攻击敌方领袖(不受反击)。这是唯一的常规伤脸途径——所有直伤法术都刻意做得稀少而昂贵(P2:胜利必须"推"出来)。
- 随从死亡即离场,亡语等效果照常结算。

### 2.6 空间机制(本作灵魂,全部服务 P1/P2)

1. **纵列压力(Lane Pressure)**:底线行的每一格都是一个潜在的失血点。5 列意味着防守方要判断"放弃哪条列"。
2. **身体阻挡**:随从互相不可穿越 → 一个 1/5 的白板"路障"在本作价值远高于炉石。
3. **地形格(可选进阶,建议第二个版本再加)**:对局地图可带 1~3 个特殊格:
   - 〔掩体〕格:格上随从受远程伤害 -1;
   - 〔裂隙〕格:回合结束对格上随从造成 1 伤害;
   - 〔补给〕格:回合结束为格上随从恢复 1 生命。
   地形让同一副卡组在不同地图上打法不同,是天然的赛季轮换抓手;但会增加首发版本的复杂度,**首发版本建议纯白板地图**。
4. **占线胜利张力**:随从到底线后不是"一次性炸弹"而是持续威胁——对方必须回防处理,由此产生攻防节奏波(P2)。

### 2.7 防拖局设计(替代炉石疲劳)

龟缩流在有地图的游戏里比炉石更容易出现,必须有硬约束:

- **牌库抽干后**:每次该抽牌时改为受到递增伤害(1, 2, 3…)——保留疲劳作为兜底。
- **压力潮汐(推荐,更主题化)**:第 8 回合起,每回合开始时,**底线行没有任何己方随从驻守的一方**受到 X 点伤害(X 随回合递增)。既惩罚全员龟缩后排,也惩罚全线空虚,机制上强迫双方"守线"——正好点题 *Hold the Line*。（先不做）
- 两者可并存,数值在测试期调。

## 3. 卡牌系统

### 3.1 卡组构成

- 卡组 **30 张**;同名卡:普通最多4张，/稀有最多 3 张,史诗2张，/传说最多 1 张。
- 构筑规则 = **1 个阵营 + 中立卡**(炉石式,身份清晰,新手理解成本低)。
- 每副卡组绑定一名**领袖**;同阵营可有多名领袖(不同技能),扩展性抓手。

### 3.2 卡牌类型

| 类型 | 说明 | 占比参考 |
|------|------|----------|
| **随从卡** | 费用 / 攻击 / 生命 + 关键词与效果;部署到底线行 | ~60% |
| **指令卡** | 一次性效果。目标可以是随从、**格子**、**行/列**、半场——"以空间为目标"的法术是本作特色(如"选择一列:该列所有敌方随从受到 2 点伤害") | ~30% |
| **装备卡** | 附着在随从上的持续增益(+攻/+移动力/授予关键词),随从死亡则消失 | ~5% |
| **战术卡(阵地)** | 放置在**空格**上的建筑类:路障、哨塔(自动远程)、陷阱(踩上触发)。不可移动、不可攻击(除非注明),用"建筑"直接表达空间价值 | ~5% |

### 3.3 费用曲线基准

- 费用 0~10,曲线体验对标炉石:1~3 费铺场抢线,4~6 费中坚对抗,7+ 费终结者。
- 因为存在"赶路回合",**高费随从普遍需要自带机动性或远程**,否则打出来 3 回合摸不到人——这是本作独有的平衡杠杆(高费卡不再只堆身材,而是堆"到场影响力")。

## 4. 关键词与机制词典

首发版本控制在 **10 个核心关键词** 以内(炉石首发约 12 个),全部围绕空间设计:

| 关键词 | 效果 | 空间意义 |
|--------|------|----------|
| **冲锋** | 部署当回合即可移动与攻击 | 调速阀,快攻核心 |
| **疾行 N** | 移动力 N(每回合可移动 N 格) | 包抄、换线 |
| **射程 N** | 见 2.5;直线 N 格内攻击,无反击 | 后排价值、掩体价值 |
| **守护** | 与其**相邻**的敌方随从必须优先攻击它(嘲讽的空间化:只护住身边,不护全场) | 站位决定保护范围 |
| **坚守** | 本回合未移动时,受到的伤害 -1 | 奖励阵地战 |
| **践踏** | 消灭敌方随从后,可立即占据其格子 | 打穿防线的钥匙 |
| **伏兵** | 不能被选为攻击/法术目标,直到它造成伤害;**但会被相邻敌人揭示** | 空间化反潜行:贴脸侦查 |
| **战吼 / 亡语** | 同炉石,部署时 / 死亡时触发 | 许多战吼以"相邻格/所在行列"为作用域 |
| **驻防** | 该随从在底线行时获得额外效果(如 +1/+1) | 奖励守线,点题 |
| **突袭** | 部署当回合可攻击(但不能移动) | 冲锋的防守版,用于反制贴线敌人 |
| **偷袭** | 攻击不受反击 |  |
| **持盾** | 免疫第一次受到的伤害 |  |

> 设计纪律:**任何新关键词必须与"格子"发生关系**,否则它属于炉石而不属于本作。

## 5. 阵营设计

首发 **4 个阵营 + 中立卡池**(第 5 阵营留作首次大版本更新的钩子)。每个阵营 = 一种鲜明的**空间人格**:

### 5.1 铁誓军团(Iron Vow)—— "墙"
- **幻想主题**:誓约骑士与堡垒工程师,断层战争中最后的正规军。
- **空间人格**:阵地战。守护、坚守、驻防关键词密度最高;拥有最多战术卡(路障、哨塔)。
- **玩法原型**:控制/防守反击。用墙耗死对方,再以高费巨物一波推线。
- **领袖技能示例**:「筑垒」(2费):使一个友方随从获得守护,直到你的下个回合。

### 5.2 荒野游群(The Wildpack)—— "风"
- **幻想主题**:兽人与掠猎兽骑手,在断层荒原上以速度为生存法则。
- **空间人格**:机动。冲锋、疾行密度最高;有换位、跳跃(无视阻挡移动)类效果。
- **玩法原型**:快攻/侧翼骚扰。多线布兵,让对方防线顾此失彼。
- **领袖技能示例**:「狩猎号角」(2费):使一个友方随从本回合移动力 +1。

### 5.3 黄昏教团(The Duskweavers)—— "焰"
- **幻想主题**:研究断层能量的放逐法师团,以牺牲换取奇迹。
- **空间人格**:以**格子/行列为目标**的法术大师。范围杀伤、地形改造(制造裂隙格)、献祭 combo。
- **玩法原型**:法术组合技/节奏。场面可以输,但一套连锁清一列。
- **领袖技能示例**:「灼痕」(2费):对一个格子及其相邻格上的所有随从造成 1 点伤害(包括友方)。

### 5.4 掘世匠会(The Undervault)—— "械"
- **幻想主题**:地底矮人与自动机械,把战争做成流水线。
- **空间人格**:远程与工程。射程关键词密度最高;炮台建筑、可组装机械(多张零件卡合成大机器人)。
- **玩法原型**:后排火力/组合成型。前排肉盾护住后排炮线,阵型即答案。
- **领袖技能示例**:「加农校准」(2费):使一个友方随从本回合射程 +1(近战随从变为射程1的远程)。

**阵营克制的自然涌现**(非硬性克制表):墙克风(机动撞墙)、风克焰(法师被贴脸)、焰克械(AOE 洗后排)、械克墙(射程隔墙点名)——由空间机制自然形成软克制环,平衡师只需微调。

## 6. 对局体验设计

### 6.1 一局的情绪曲线(目标 8~12 分钟)

| 阶段 | 回合 | 玩家心流 |
|------|------|----------|
| 布阵期 | 1~3 | "我这局走哪几条列?" 低压力,铺场与探路 |
| 接触期 | 4~6 | 前线行接火,第一波交换;开始出现"弃哪条列"的痛苦决策 |
| 拉锯期 | 7~9 | 攻防转换,伤脸开始出现,压力潮汐临近 |
| 终结期 | 10~13 | 一方防线洞开或大哥落地,斩杀窗口出现 |

### 6.2 决策密度控制(P4)

- 单回合"有效决策点"目标 3~5 个(出什么牌 × 部署哪列 × 谁动谁打)。
- **回合计时**:75 秒基准 + 炉石式引线烧绳;移动操作全部拖拽/点选两步完成,无多余确认。
- 场上随从数天然被 20 格与阻挡规则约束,不会出现炉石 7 站位 × 复杂结算的爆炸场面。

### 6.3 反挫败设计

- 被斩杀前一定看得到威胁(随从要走到底线才能打脸)→ **本作几乎没有"空气斩杀"**,挫败感天然低于炉石。这是核心体验优势,营销时要讲。
- 随机性纪律:效果随机(如"随机一列")允许,**结算随机(如 50% miss)禁止**。

## 7. 游戏模式

| 模式 | 说明 | 优先级 |
|------|------|--------|
| **排位对战** | 天梯,月度赛季,段位保护 | P0(首发) |
| **休闲对战** | 无段位压力,测卡组 | P0 |
| **好友对战 + 观战** | 私人房间;观战延迟 2 回合防作弊 | P0(联机游戏的社交底座) |
| **战线远征(PvE Roguelike)** | 单人爬塔:预组卡组+局中拾牌,打 AI 关卡;兼任新手教学与断网可玩内容 | P1(首发后 1~2 个月) |
| **轮抽(竞技场)** | 三选一组卡组,赢场兑奖 | P1 |
| **异变模式** | 每周规则变体(如 3×5 快速地图、地形狂潮) | P2 |

## 8. 体验循环

### 8.1 循环分层

```
┌─ 单局循环 (8~12 分钟) ─────────────────────────┐
│ 布阵 → 接触 → 拉锯 → 终结 → 结算(经验/任务进度) │
└──────────────┬───────────────────────────────┘
┌─ 单日循环 (20~40 分钟) ──────▼─────────────────┐
│ 每日任务(3个,可换1个)→ 打 2~4 局 → 领奖励      │
│ 奖励 = 卡包碎片 / 金币 / 赛季经验                │
└──────────────┬───────────────────────────────┘
┌─ 每周循环 ────▼───────────────────────────────┐
│ 周常任务 + 异变模式 + 天梯段位结算点              │
└──────────────┬───────────────────────────────┘
┌─ 赛季循环 (1~2个月) ─▼────────────────────────┐
│ 新卡 mini 包 / 平衡补丁 / 段位重置 / 赛季通行证    │
│ 赛季叙事:战线地图上的阵营战争推进(见 9.4)        │
└──────────────┬───────────────────────────────┘
┌─ 长线循环 (年) ▼──────────────────────────────┐
│ 大版本 = 新阵营/新领袖/新机制关键词 + 世界观章节    │
└───────────────────────────────────────────────┘
```

### 8.2 收集与养成

- **获取**:卡包(随机)+ 合成系统(分解重复卡→尘→指定合成,直接沿用炉石成熟方案)+ 成就送卡。
- **付费边界(如走 F2P)**:只卖卡包/通行证/皮肤(领袖皮肤、棋子皮肤、**棋盘皮肤**——网格地图让"棋盘"成为天然皮肤位,炉石没有的商业化点)。**不卖数值、不卖体力。**
- 新手 7 日签到送一套完整的"铁誓军团"入门卡组,保证第 1 天就能玩到成型套路。

### 8.3 社交与对抗生态

- 好友、观战、卡组代码分享(文本串,一键导入)。
- 赛季末"阵营战争"结算:全服玩家的排位胜场按阵营累计,胜出阵营玩家获得专属棋盘皮肤 → 把天梯个人行为聚合成世界观事件(见 9.4)。

## 9. 世界观:断层战线

### 9.1 高概念

> 世界"艾恩大陆"在三百年前被一场灾变撕开了一道横贯东西的**断层(The Sunder)**——一条不断渗出原初能量"辉尘"的大地裂缝。辉尘既是最强的资源,也是缓慢扩散的灾难。四大势力沿断层两岸筑起绵延的防线,既为了争夺辉尘,也为了阻止裂缝对面的东西过来。
>
> 每一场对局,就是战线上某一段阵地的一次攻防。**Hold the Line** 既是玩法(守住底线),也是世界观的全部主题(所有人都在守自己的线)。

### 9.2 世界观与玩法的互文(这是重点,不是贴皮)

| 玩法元素 | 世界观解释 |
|----------|-----------|
| 网格战场 | 断层沿岸的"阵地段",战壕与工事把战场切成格子 |
| 费用水晶 | **辉尘**:双方都在从断层抽取能量,越打越多(回合数↑) |
| 随从部署在底线 | 部队从己方战壕投入战场 |
| 到达敌方底线打本体 | 攻入敌方指挥部 |
| 压力潮汐(2.7) | 断层能量周期性涨潮,冲刷防线空虚的一方 |
| 卡牌 | 指挥官(玩家)手中的**作战预案**;抽牌 = 后方参谋部的临场调度 |
| 四阵营 | 沿断层四段防线的四种文明应答:铸墙、游猎、驭焰、掘地 |

### 9.3 玩家身份

玩家是**战线指挥官(Warden)**,领袖卡是你麾下的传奇战地将领。天梯段位命名走军衔体系:列兵→军士→尉官→校官→将官→**执线者(传说)**。

### 9.4 活的战线(赛季叙事)

- 主界面是一张**断层战线全景图**,四阵营各占区段。
- 每个赛季 = 一次"战役":全服玩家排位胜场按阵营累计,赛季末战线图上的阵营边界**真实移动**,并决定下赛季的叙事文本与奖励皮肤。
- 大版本揭露断层"对面"的存在(第 5 阵营伏笔:来自裂缝彼端的**彼岸潮**),世界观自带一个可以长期吊胃口的悬念引擎。

### 9.5 基调与叙事载体

- 基调:**战地浪漫主义**——残酷战线上的兵种小人物群像,幽默感来自卡牌语音与卡面轶事(参考炉石的语调,但少些插科打诨、多些战壕黑色幽默)。
- 载体:卡面描述(风味文本)、领袖间的开场对白、PvE 远征模式章节剧情。**不做过场动画长剧情**,叙事全部碎片化、可跳过。

## 10. 新手引导与留存设计

1. **教学 = 3 关 PvE 剧本**(15 分钟):第 1 关只教"部署与移动"(不给法术),第 2 关教战斗与守护,第 3 关完整对局打剧情 AI。
2. **新手保护匹配**:前 10 场排位只匹配同池新手与教学 Bot(Bot 有拟人昵称,胜率控制在 55% 给玩家)。
3. **入门卡组即战力**:送的铁誓卡组必须能上到中段位,避免"没卡不能玩"的第一周流失。
4. **败局学习**:战报回放 + "本局关键回合"标记(空间类游戏,复盘价值远高于炉石,做好了就是留存利器)。

## 11. 平衡性框架

- **数值锚点**:1 费白板基准 = 1/2 或 2/1;每 +1 费 ≈ +2 总身材;关键词按"等效身材"计价(冲锋≈1费、射程2≈1.5费、疾行≈1费……),建立**关键词价目表**并随实测更新。
- **空间价值量化**:移动力与射程是本作独有的平衡轴。原则:**攻击力可以贵,机动性必须更贵**——机动膨胀比数值膨胀更容易杀死游戏(所有人都秒到脸,地图就死了)。
- **平衡节奏**:赛季中只做增强性微调,数值刀集中在赛季末;每张卡改动附带设计师短注(社区信任感)。
- **数据看板**:胜率、出场率、"该卡在场时平均占格时长"(本作特有指标,衡量空间贡献)。

## 12. 待定设计问题清单

需要在原型期用实际对局回答的问题(按优先级):

1. **4×5 还是 3×5?** 纸面原型(实体卡+棋盘)一天就能验证,建议数字化之前先做。
2. **移动与攻击是否应合并为"1 行动点"?**(即每回合移动*或*攻击二选一)——会大幅降低快攻强度、拉慢节奏,备选方案。
3. **守护(相邻嘲讽)是否过弱?** 若防守撑不住快攻,升级为"同列嘲讽"。
4. **部署是否只限底线行?** 备选:允许部署到己方半场任意空格但+1费("空投")作为关键词而非通则。
5. **压力潮汐的起始回合与数值。**
6. **领袖技能是否首发就要?** 砍掉可降低平衡复杂度,但阵营身份感会弱。倾向保留。
