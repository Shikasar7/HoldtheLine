## 1. 数值锚点(平衡框架落地)

所有身材按"价值点"估算,允许 ±0.5 浮动,阵营特色卡允许 +1 溢价(快攻卡在快攻阵营略超模是健康的):

- **价值点 = 1.3 × 攻击 + 1.0 × 生命 + 关键词税 + 效果税**
- **费用预算 ≈ 2.2 × 费用 + 1.2**(1费≈3.4,2费≈5.6,3费≈7.8,4费≈10.0,5费≈12.2,6费≈14.4,7费≈16.6,8费≈18.8)

**关键词税**(初版,由模拟器和实测修正):

| 关键词 | 税 | 说明 |
|---|---|---|
| 冲锋 | 2.2 | 本作最贵——省 2~3 回合赶路 |
| 突袭 | 1.2 | 只能原地打,防守向 |
| 疾行(每+1移动力) | 1.5 | 机动必须比数值贵(GDD §11) |
| 射程 2 / 3 | 2.2 / 3.2 | 免反击含在内 |
| 守护 | 1.2 | 0 攻纯墙另有 -1 功能折价 |
| 坚守 | 1.2 | |
| 践踏 | 1.0 | |
| 偷袭 | 1.8 | |
| 持盾 | 1.8 | |
| 驻防 | 1.2 | 统一定义:**在己方底线行时 +1/+1**(见 §5) |
| 跃障 | 1.8 | 仅 1 张实验卡(见 §5) |

**效果税**:抽1≈2.2,治疗2≈1.2,单体2伤≈2.4,+1/+1≈2.3,+0/+2≈1.8,本回合移动+1≈1.0。

## 5. P2 引擎扩展规格(给 Opus 的施工单)

本表用到但引擎尚未实现的能力,**按此清单扩展,不得即兴发明**。EffectSpec 形状的唯一变更:新增可选字段 `duration`(本次 Fable 批准)。

| # | 扩展 | 规格 | 用到的卡 |
|---|---|---|---|
| 1 | action: `heal` | 恢复生命,不超过 MaxHp | field_medic |
| 2 | action: `grant_keyword` | 为目标附加关键词;配合 `duration` | entrench, shield_wall, pounce, 领袖「筑垒」 |
| 3 | action: `summon` | 在己方底线行空格召唤指定 card_id ×N(格不够则少召) | pack_call |
| 4 | action: `move_bonus` | 本回合移动力 +N(实现:MovementUsed 减免或临时上限) | howler, hunt_signal, 领袖「狩猎号角」, 军令硬币B选项(暂缓) |
| 5 | selector: `column_enemies` | 以 PlayCardCommand.TargetCell 所在**列**为域选敌方 | focused_barrage |
| 6 | selector: `allies_home_row` | 己方底线行全部友方 | hold_the_line |
| 7 | selector: `all_allies` | 全部友方随从 | blood_scent |
| 8 | 目标过滤: `own_half` | target_unit 限定在施法者半场(row 0-1 / 2-3) | overline_execution |
| 9 | **duration 系统** | EffectSpec.duration: `permanent`(默认) / `end_of_turn` / `your_next_turn`;回合边界统一结算过期 | pounce、move_bonus 系、「筑垒」 |
| 10 | **驻防(Garrison)机制** | 统一规则:**位于己方底线行时 +1/+1**;部署/移动/践踏占格后重算 | garrison_guard, fortress_lord |
| 11 | **跃障(Leap)关键词** | 移动时可选择同行/同列**距离 2** 的空格,跨过中间的随从,消耗 1 移动力 | rift_leaper(仅此 1 张)|
| 12 | **领袖技能** | UseLeaderSkillCommand:2 费,每回合 1 次;技能=EffectSpec 复用 | 两名领袖 |

> 砍单许可:若 P2 进度吃紧,**#11 跃障优先砍**(rift_leaper 改为"疾行 3");其余不可砍。
> 军令硬币维持单效果(+1 费),GDD 的"二选一"模式化留给正式版。

## 8. 审稿引导

重点审这几处,其余数值交给模拟器迭代:

1. **纵列齐射(3费全列1伤)**——签名牌强度是否给够?弱了没人用,签名机制就哑了。
2. **越线者死**的"只打己方半场"限制——你是否喜欢这种"防守反击"的表达?
3. **跃障实验卡**留不留(P2 最贵的一项)。
4. **驻防统一 +1/+1** vs 每卡自定义——我选了统一,理由是规则可背诵。
5. 传说卡的幻想感:雷诺克/加兰/卡鲁姆三个名字与人设是否合口味。
